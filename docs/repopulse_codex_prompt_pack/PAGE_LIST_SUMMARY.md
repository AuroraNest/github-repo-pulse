# RepoPulse 页面列表速览

| 页面 | 作用 | MVP 优先级 |
|---|---|---|
| Setup / Connect GitHub | 首次配置 GitHub Token、选择仓库、设置同步时间 | P0 |
| Overview | 所有仓库总览，展示总 stars、forks、downloads、visitors、clones、活动摘要 | P0 |
| Repositories | 管理被监控仓库，搜索、筛选、排序、手动同步 | P0 |
| Repository Detail | 单仓库分析，展示趋势、流量、下载、热门页面、来源网站 | P0 |
| Releases / Downloads | Release asset 下载统计，累计下载、今日新增、下载排行 | P0 |
| Reports | 每日/每周/月度报告，支持规则总结、AI 总结、导出 | P0 |
| Settings | GitHub 连接、同步计划、数据保留、AI、通知、备份、安全 | P0 |
| Activity / Sync Logs | 同步记录、失败原因、rate limit、重试 | P1 |
| Alerts / Automation | 下载暴涨、同步失败、stars 增长等告警规则 | P1 |
| Integrations | Email、Webhook、Telegram、Discord、Slack、AI Provider 等集成 | P2 |
| Data Export / Backup | CSV/JSON/Markdown/PDF 导出、数据库备份 | P2，可先合并进 Settings |
