# Phase 05：开源发布打磨

## 给 Codex 的阶段提示词

把 RepoPulse 打磨成可开源发布的项目。

必须完成：

1. README 完整。
2. LICENSE。
3. SECURITY.md。
4. CONTRIBUTING.md。
5. .env.example。
6. Docker Compose 一键启动。
7. GitHub Actions CI。
8. Demo screenshots 文档。
9. 安全审查。
10. Release notes 模板。

## README 必须说明

- RepoPulse 是什么。
- 为什么 GitHub Traffic 需要长期保存。
- 快速开始。
- GitHub Token 权限。
- AI 可选。
- 数据隐私。
- 常见问题。

## 验收

- 新用户 clone 后 10 分钟内能跑起来。
- 没有 token 能看 demo。
- 有 token 能采集真实数据。
- Docker 部署不需要复杂手动步骤。
