# Phase 03：核心 UI 页面实现

## 给 Codex 的阶段提示词

基于真实数据库数据，实现核心 UI 页面。

页面：

1. Overview。
2. Repositories。
3. Repository Detail。
4. Releases / Downloads。
5. Reports。
6. Settings。

## 要求

- 所有图表从 API 数据获取。
- 所有表格支持 loading/empty/error。
- Overview KPI 要正确计算 delta。
- Repositories 支持搜索、过滤、排序。
- Repository Detail 支持 tabs 和时间范围。
- Releases 支持 asset 搜索和排序。
- Settings 能保存配置。

## 验收

- demo 数据和真实数据都能展示。
- UI 与参考图风格一致。
- 移动端不崩。
- 所有关键按钮有交互。
